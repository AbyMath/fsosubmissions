# Bloglist Frontend

Frontend for the **Bloglist** application built with **React** as part of the **Full Stack Open** course.

## Features

- User authentication
- Create, like, comment on, and delete blogs
- View all users and their blogs
- Client-side routing with React Router
- Material UI interface

## Tech Stack

- React
- Vite
- React Router
- Axios
- Material UI
- Vitest
- Playwright

## Course

Developed as part of **Full Stack Open – Part 5: Testing React Apps**.
